Plantilla en Android Studio en el que se ingresa una cantidad de consumo y a partir de el selecciona un porcentaje de propina puede ser del (10%, 15% y 20%), 
ademas de un switch para determinar si se requiere o no de redondear la propina y un boton para hacer el calculo, para finalizar un textview para mostrar el resultado de la operacion.
